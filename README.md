# St. Louis Live Crime
A live map of crime in the city of St. Louis, MO.

[![Screenshot](http://i.imgur.com/Qjrjjy6.png)](http://gavinr.github.io/stl-live-crime/)

[View Here](http://gavinr.github.io/stl-live-crime/)

## Techologies

* ArcGIS JS API 4.0
* ES2015 (Babel transpiler)
* Gulp
* Stylus

## Get Started

* `npm install`
* `gulp watch`
